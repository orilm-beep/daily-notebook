      First time? How to set up Supabase (3 minutes)
      1. Go to supabase.com → sign up (free) → New project.
      2. Open SQL Editor, paste this and click Run:
      create table if not exists tasks ( id text primary key, text text, project text, done boolean default false, created_at timestamptz, completed_at timestamptz, updated_at timestamptz, deleted boolean default false ); alter table tasks enable row level security; create policy "personal access" on tasks for all to anon using (true) with check (true);
      1. Go to Project Settings → API. Copy the Project URL and the anon public key into the fields above.
      2. Click Connect & Sync. Repeat step 3–4 on your iPhone to share the list.
